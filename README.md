## TERMCHAT - ONLY ARCH+UBUNTU 4NOW 
A very bad terminal messaging app i made just to test my skills in C.
You probably shouldn't use (or even download) this app. It's purely made
for me and a couple of friends just to have fun in the computer science lab
in our school. Part of the code HAS been written with AI. It's just a fun 
project and I don't recommend using it. I'm just happy it works. 

## PORT 
default port is set to 5000, you don't have to add it yourself. Edit .c files
to change port 

## CONNECT
termchat connect <ip>

## LISTEN
termchat listen 

## RUNNING THE PROGRAM 
git clone https://github.com/BlitzerFuse/termchat && cd termchat && ./bootstrap.sh

## NOTICE
you may need to give execute perms to bootstrap.sh and add dependencies manually.
